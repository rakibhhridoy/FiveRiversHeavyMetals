METHOD 3: PYTHON SCRIPT (Advanced)
==================================

Create a Python script with visualization functions:

```python
import rasterio
import numpy as np
import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

# Load raster
raster_path = '/Users/rakibhhridoy/Five_Rivers/gis/AlphaEarth/AlphaEarth_64_Bands.tif'

with rasterio.open(raster_path) as src:
    data = src.read()  # Load all 64 bands
    h, w = data.shape[1], data.shape[2]

    # 1. VIEW SINGLE BAND
    band_1 = src.read(1)
    plt.imshow(band_1, cmap='viridis')
    plt.title('Band 1')
    plt.colorbar()
    plt.savefig('band_1.png')

    # 2. CREATE PCA COMPOSITE (RGB)
    data_flat = data.transpose(1,2,0).reshape(-1, 64)
    scaler = StandardScaler()
    data_scaled = scaler.fit_transform(data_flat)

    pca = PCA(n_components=3)
    pca_data = pca.fit_transform(data_scaled)
    pca_img = pca_data.reshape(h, w, 3)

    # Normalize to 0-1
    pca_norm = (pca_img - pca_img.min()) / (pca_img.max() - pca_img.min())

    plt.imshow(pca_norm)
    plt.title('PCA Composite (3 Principal Components)')
    plt.savefig('pca_composite.png')
```

================================================================================
UNDERSTANDING THE VISUALIZATIONS
================================================================================

1. all_64_bands.png
   - 8Ã—8 grid showing all 64 AlphaEarth bands
   - Each band represents different satellite data characteristics
   - Bands with similar patterns are correlated
   - Use this to identify dominant features

2. pca_composite.png
   - RGB image created from 3 principal components
   - Reduces 64 dimensions to 3 (Red, Green, Blue)
   - Preserves ~80-90% of variance
   - Best for quick visualization and interpretation

3. band_statistics.png
   - Mean value per band (top-left)
   - Standard deviation per band (top-right)
   - Min/Max ranges (bottom-left)
   - Distribution histogram (bottom-right)
   - Identify bands with high/low variance

4. band_correlation_full.png
   - 64Ã—64 correlation matrix
   - Shows relationships between all bands
   - Warm colors (red) = positive correlation
   - Cool colors (blue) = negative correlation
   - Identify redundant bands

5. selected_bands.png
   - Detailed view of key bands (1, 2, 3, 4, 5, 10, 20, 30, 40, 50)
   - Good for publication figures
   - Shows spatial variation across bands

6. AlphaEarth_PCA3.tif
   - 3-band GeoTIFF file
   - Can be imported into ArcGIS, QGIS, etc.
   - Use with GIS software for geospatial analysis

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'rasterio'"
Solution: pip3 install rasterio

Problem: "File not found" error
Solution: Make sure AlphaEarth file is in correct location:
  /Users/rakibhhridoy/Five_Rivers/gis/AlphaEarth/AlphaEarth_64_Bands.tif

Problem: Earth Engine export is very slow
Solution:
  - Reduce area of interest (AOI) size
  - Use smaller spatial resolution (e.g., 20m instead of 10m)
  - Export in multiple tiles

Problem: Out of memory error
Solution:
  - Close other applications
  - Increase available RAM
  - Process smaller area
  - Use GIS software (QGIS) for large rasters instead

Problem: PCA composite looks blank
Solution:
  - Check if raster values are in expected range
  - Verify all 64 bands are present
  - Use percentile clipping (2nd to 98th)

================================================================================
INTERPRETATION TIPS
================================================================================

1. Band Interpretation:
   - Each band captures different satellite information
   - Optical bands: visible light characteristics
   - SAR bands: radar scattering patterns
   - Vegetation indices: plant health and density
   - Topography: elevation and slopes

2. Correlation Analysis:
   - Highly correlated bands are redundant
   - Uncorrelated bands add independent information
   - Use correlation to select subset of bands

3. PCA Results:
   - PC1 usually captures ~30-40% of variance
   - PC1+PC2 typically ~60-70% of variance
   - PC1+PC2+PC3 usually ~80-90% of variance
   - Remaining variance spread across PC4-PC64

4. Feature Selection:
   - Bands with high variance are important
   - Low-variance bands are mostly noise
   - Consider correlation when selecting bands
   - Use PCA loadings to identify key bands
