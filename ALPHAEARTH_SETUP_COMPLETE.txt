HOW TO GET STARTED
================================================================================

STEP 1: VERIFY PREREQUISITES (5 minutes)

  Check you have:
  - Google Earth Engine account (free at earthengine.google.com)
  - earthengine-api installed: pip install earthengine-api
  - Authentication: python -m earthengine authenticate

  Verify all dependencies:
    python3 RUN_ALPHAEARTH_COMPARATIVE_ANALYSIS.py
    (This will check all prerequisites)

STEP 2: CHOOSE YOUR APPROACH

  OPTION A: Quick Test (Fast, 2-4 hours)
    ./run_alphaearth.sh test-rainy
    - Tests 1 model with Option B on Rainy season
    - Fastest way to see if everything works

  OPTION B: Recommended (44-88 hours)
    ./run_alphaearth.sh rainy-b
    ./run_alphaearth.sh winter-b
    - Tests all 11 models with Option B (best choice)
    - Complete for both seasons

  OPTION C: Full Analysis (4-6 weeks)
    ./run_alphaearth.sh full
    - Tests all 11 models Ã— 4 options Ã— 2 seasons
    - Complete comparative analysis

STEP 3: RUN DATA PREPARATION

  Before running models, prepare data:
    ./run_alphaearth.sh data-both

  This extracts AlphaEarth embeddings and creates:
  - Option_A_RainyAE.csv
  - Option_B_RainyAE.csv
  - Option_C_RainyAE.csv
  - Option_D_RainyAE.csv
  - pca_alpha_earth.pkl
  (and same for Winter season)

STEP 4: RUN MODELS

  After data preparation, run models:
    ./run_alphaearth.sh quick    # Fastest
    ./run_alphaearth.sh rainy-b  # All models, Option B
    ./run_alphaearth.sh full     # Complete analysis

STEP 5: CHECK RESULTS

  Results saved to:
  - /gis/SedimentRainyAE/       (outputs, models, features)
  - /gis/SedimentWinterAE/      (outputs, models, features)

  Model performance metrics in notebook outputs
  Feature importance in */FeatureImportance/ directories
  Saved models in */models/ directories

OPTION B: Add to Features (RECOMMENDED)
  - Keep all existing features
  - Add 64 AlphaEarth bands
  - CNN channels: 86-88
  - Expected gain: +3-6% (rainy), +2-4% (winter)
  - Best balance of improvement vs complexity

OPTION C: PCA-Reduced
  - Add PCA-reduced AlphaEarth (20 components)
  - CNN channels: 42-44
  - Expected gain: +2-4% (rainy), +1-3% (winter)
  - Faster training, lower memory

OPTION D: MLP Enhancement
  - Add summary statistics to MLP input only
  - CNN unchanged
  - Expected gain: +0.5-2% (rainy), +0.3-1% (winter)
  - Minimal code changes
